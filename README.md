# Evidence zásilek objektově

požadované API (naprogramuj všechno):
- registruj zasilku()
- zasilka prevzata() – odesilatel odnesl na pickup-point
- zasilka ztracena()
- zasilka na ceste() – k prijemci
- zasilka dorucena()
- zasilka vracena()
- historie zasilky()
- zasilka info()

Extra iniciativa bude hodnocena extra vahou. Defaultní váha je 2.

## Nezapomeň
- čitelnost kódu má přednost přes super-efektivitou (ale mapa je v řadě případů daleko lepší než seznam)
- dodržet _pravopis_
- domysli správně signatury metod, aby dávaly smysl
- vymyslí **optimální datový model**, aby se ti snadno a čitelně programovalo
- typové anotace jsou nutnost: červená podtržení v IDE by neměla existovat
- použij vše co ses naučil
